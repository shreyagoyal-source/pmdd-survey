# Menstrual Health & Nutrition Survey

This is a simple interactive web survey that asks about menstrual symptoms, diet changes, and nutritional awareness.

## How to run

1. Open `index.html` in a web browser (double-click or right-click → Open With).
2. Use the **Next →** button to move through each question.
3. At the end, a summary of your responses will be shown.

## Notes

- The survey runs entirely in your browser with no data sent anywhere.
- To share with others, upload the folder to a static hosting provider (GitHub Pages, Netlify, Vercel) or share the folder directly.
